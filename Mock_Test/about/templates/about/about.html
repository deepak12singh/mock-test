About Our Project

Exam Prep Site is a student-focused web application designed to support aspirants preparing for exams like CUET UG, CUET PG, and more in the future such as SSC, UPSC, and Bank exams. This platform offers a smooth and free mock testing experience with performance tracking and essential exam resources.

Our Mission: To provide a free, accessible, and powerful preparation platform for every student aiming for competitive exams, regardless of their background or location.

Key Features

  • Free subject-wise mock tests for CUET UG and PG
  • Performance tracking and analysis for every test attempt
  • Latest notifications, syllabus, FAQs, and exam tips
  • Clean user interface with easy navigation
  • Admin panel for managing content, questions, and users

Current Coverage

The app currently supports:

  • CUET UG - Major subjects with syllabus and practice tests
  • CUET PG - Streams like English, Hindi, MCA, MBA, M.Com, and MA

Tech Stack

  • Frontend: HTML, CSS, JavaScript
  • Backend: Django (Python)
  • Database: SQLite (development), with future integration to cloud DB
  • Hosting: Google Cloud Platform

Future Enhancements

  • Adding mock tests for more competitive exams like SSC, UPSC, and Banking
  • Introducing a mobile-friendly version of the site
  • AI-driven question suggestions and adaptive learning modules

Why Use Our Platform?

We created this platform out of the need for a truly student-centric exam preparation space. Our system is free, user-friendly, and tailored to the actual exam formats. We're constantly evolving based on student feedback and competitive exam trends.
